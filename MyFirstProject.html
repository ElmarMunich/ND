Atom

I have chosen atom as my "texteditor". There are some useful shortcuts:

  • Creating the outline of an HTML document. First install the emmet plug-in emmet plug-in
    then you can use !->| to create the html5 outline.
  • Preview code in atom. First install the preview plug-in PreView plug-in
    then you can use ctrl+^+h to get a preview.
